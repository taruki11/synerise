## 6. Why Large Artifacts Are Not Tracked Here

This is a public portfolio repository, so the following are intentionally kept out of Git tracking:

- raw source data
- processed feature tables
- large generated CSV and JSON result files
- Word and HTML export files

This keeps the repository cleaner, easier to browse, and more suitable for public presentation.
